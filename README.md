# spraying-robot
AI-based spraying robot for precision agriculture

### Dataset Source
Kaggle: PlantVillage Dataset  
https://www.kaggle.com/datasets/emmarex/plantdisease

## Technologies Used
- Python
- TensorFlow
- OpenCV
- CNN
- Raspberry Pi / Arduino (conceptual)

## Working
1. Capture plant image using camera
2. Classify plant as healthy or diseased using CNN
3. Activate sprayer only for diseased plants

## Output
Targeted spraying reduces pesticide usage and supports sustainable farming.
